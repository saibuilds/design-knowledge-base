AI Component Marketplace

21st.dev

AI-powered component marketplace for shipping UI 10x faster. Search components, install via shadcn registry protocol, and integrate directly with Cursor IDE via the Magic MCP server. Built for the AI-assisted development era.

Official Site Docs Browse Components

What is 21st.dev?

A marketplace where developers publish and discover UI components. Uses the shadcn registry spec so every component installs with one command.

Key differentiator: 21st.dev includes a Magic MCP server that integrates directly with Cursor, Windsurf, and other AI IDEs. When you ask your AI assistant to "add a hero section", it searches 21st.dev and installs the best match automatically — no manual browsing.

Format: All components follow the shadcn registry format. Install with npx shadcn@latest add [url] or via the Magic MCP tool.

Installing Components

bash — CLI install
# Search and install from 21st.dev
npx shadcn@latest add "https://21st.dev/r/shadcn/animated-hero"
npx shadcn@latest add "https://21st.dev/r/shadcn/navigation-menu"
npx shadcn@latest add "https://21st.dev/r/shadcn/pricing-section"
npx shadcn@latest add "https://21st.dev/r/shadcn/testimonials"
npx shadcn@latest add "https://21st.dev/r/shadcn/footer"
npx shadcn@latest add "https://21st.dev/r/shadcn/feature-section"

Magic MCP — Cursor Integration

The Magic MCP server lets your AI assistant (Cursor, Windsurf) browse and install 21st.dev components on command.

json — .cursor/mcp.json
{
  "mcpServers": {
    "21st-magic": {
      "command": "npx",
      "args": ["-y", "@21st-dev/magic@latest", "API_KEY=your-key-here"]
    }
  }
}
text — Cursor chat prompts
// In Cursor chat with Magic MCP active:
"Add a hero section with gradient text and CTA buttons"
"Install a pricing table with 3 tiers"
"Add a features bento grid section"
"Install a dark theme testimonials carousel"
"Add an animated navigation menu with dropdowns"

Component Categories

Sections: Hero, Features, Pricing, Testimonials, FAQ, CTA, Footer, About
Navigation: Navbar, Sidebar, Breadcrumb, Mobile menu
Marketing: Social proof, Logo ticker, Stats, Team grid
Forms: Contact, Newsletter, Auth pages, Multi-step
Overlays: Modal, Sheet, Command palette, Notifications
Data: Tables, Charts, Dashboards, Analytics widgets
Animations: Animated counters, Reveal effects, Loading states

Each component is standalone (includes all required sub-components), responsive, and dark-mode compatible.

Publishing Components

bash — publish your own component
# Login to 21st.dev
npx @21st-dev/cli login

# Publish a component from your project
npx @21st-dev/cli publish components/ui/my-component.tsx

# The component gets a registry URL:
# https://21st.dev/r/your-username/my-component
