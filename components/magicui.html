Setup

Requires shadcn/ui already initialized. Magic UI uses the same registry protocol.

bash — install any Magic UI component
# Prerequisite: shadcn/ui must be initialized
npx shadcn@latest init

# Install individual components
npx shadcn@latest add "https://magicui.design/r/marquee"
npx shadcn@latest add "https://magicui.design/r/animated-list"

Marquee

Infinite scrolling ticker — perfect for logos, testimonials, and feature lists. CSS animation with pause-on-hover.

bash
npx shadcn@latest add "https://magicui.design/r/marquee"
jsx
import Marquee from '@/components/ui/marquee';

const logos = ['Vercel', 'Stripe', 'Linear', 'Raycast', 'Clerk'];

export function LogoTicker() {
  return (
    <Marquee
      pauseOnHover    // pause animation on mouse hover
      reverse={false} // direction: false = left, true = right
      speed={30}      // CSS animation duration in seconds
      className="py-4"
    >
      {logos.map(logo => (
        <div key={logo} className="mx-8 text-muted-foreground font-semibold text-lg">
          {logo}
        </div>
      ))}
    </Marquee>
  );
}

// Double row (common pattern)
export function TestimonialMarquee() {
  return (
    <div className="flex flex-col gap-4">
      <Marquee pauseOnHover>
        {topRow.map(t => <TestimonialCard key={t.id} {...t} />)}
      </Marquee>
      <Marquee pauseOnHover reverse>
        {bottomRow.map(t => <TestimonialCard key={t.id} {...t} />)}
      </Marquee>
    </div>
  );
}

AnimatedList

Staggered list with smooth spring animations — great for activity feeds, notifications, and feature lists.

jsx
import { AnimatedList } from '@/components/ui/animated-list';

const notifications = [
  { name: 'Payment received', description: '$299.00', icon: '💳', color: '#22c55e', time: '2s ago' },
  { name: 'New subscriber', description: 'john@example.com', icon: '🎉', color: '#3b82f6', time: '5s ago' },
  { name: 'Deploy complete', description: 'v2.4.1 → Production', icon: '🚀', color: '#8b5cf6', time: '12s ago' },
];

export function NotificationFeed() {
  return (
    <AnimatedList delay={1500}> {/* delay in ms between items */}
      {notifications.map(n => (
        <div key={n.name} className="flex items-center gap-3 rounded-xl bg-card p-4 border">
          <span className="text-2xl">{n.icon}</span>
          <div>
            <p className="font-semibold text-sm">{n.name}</p>
            <p className="text-muted-foreground text-xs">{n.description}</p>
          </div>
          <span className="ml-auto text-xs text-muted-foreground">{n.time}</span>
        </div>
      ))}
    </AnimatedList>
  );
}

Bento Grid

Apple-inspired feature grid with varying cell sizes. The most common landing page pattern in 2024-2026.

jsx
import { BentoCard, BentoGrid } from '@/components/ui/bento-grid';

const features = [
  {
    Icon: CodeIcon,
    name: 'Ship faster',
    description: 'Copy-paste components into your codebase.',
    href: '#',
    cta: 'Learn more',
    background: <AnimatedBeam className="absolute inset-0" />,
    className: 'lg:row-start-1 lg:row-end-4 lg:col-start-2 lg:col-end-3', // wide cell
  },
  {
    Icon: ZapIcon,
    name: 'Zero config',
    description: 'Works out of the box with your existing setup.',
    className: 'lg:col-start-1 lg:col-end-2 lg:row-start-1 lg:row-end-3',
  },
];

export function FeaturesSection() {
  return (
    <BentoGrid className="lg:grid-rows-3">
      {features.map(f => (
        <BentoCard key={f.name} {...f} />
      ))}
    </BentoGrid>
  );
}

Background Effects

jsx
import { DotPattern } from '@/components/ui/dot-pattern';
import { GridPattern } from '@/components/ui/grid-pattern';
import { Particles } from '@/components/ui/particles';
import { Meteors } from '@/components/ui/meteors';
import { RetroGrid } from '@/components/ui/retro-grid';
import { Ripple } from '@/components/ui/ripple';
import { cn } from '@/lib/utils';

// Dot pattern background
<DotPattern
  className={cn('[mask-image:radial-gradient(300px_circle_at_center,white,transparent)]')}
/>

// Grid pattern
<GridPattern
  numSquares={30}
  maxOpacity={0.1}
  duration={3}
  className="inset-x-0 inset-y-[-30%] h-[200%] skew-y-12"
/>

// Floating particles
<Particles quantity={100} ease={80} color="#ffffff" />

// Shooting stars / meteors
<Meteors number={20} />

// Perspective grid (footer/hero)
<RetroGrid />

// Ripple circles (hero behind CTA)
<Ripple />

// Hero section pattern
<section className="relative overflow-hidden">
  <DotPattern className="absolute inset-0 opacity-50 [mask-image:radial-gradient(ellipse_at_center,white,transparent)]" />
  <div className="relative z-10">{/* content */}</div>
</section>

GitHub README Notes

From magicuidesign/magicui README: "Magic UI is a collection of re-usable components that you can copy and paste into your web apps. It primarily features components, blocks, and templates geared towards creating landing pages and user-facing marketing materials."

Built on: shadcn/ui registry protocol + Framer Motion + Tailwind CSS + Radix UI

Requirements: Next.js (or any React framework) + shadcn/ui initialized + Tailwind CSS + Framer Motion

Stars: 18k+ GitHub stars (2025)

Pricing: Free open-source components + paid Pro templates for complete landing page sections
