Installation Guide

Prerequisites

Both libraries use the shadcn CLI which adds the component files directly into your project (no black-box node_modules).

1. Initialize shadcn (if not done)

npx shadcn@latest init

2. Install framer-motion (required by most components)

npm install motion

3. Add components as needed

# MagicUI pnpm dlx shadcn@latest add @magicui/animated-beam pnpm dlx shadcn@latest add @magicui/marquee pnpm dlx shadcn@latest add @magicui/number-ticker pnpm dlx shadcn@latest add @magicui/shimmer-button pnpm dlx shadcn@latest add @magicui/animated-gradient-text pnpm dlx shadcn@latest add @magicui/bento-grid pnpm dlx shadcn@latest add @magicui/particles # Aceternity UI npx shadcn@latest add @aceternity/3d-card npx shadcn@latest add @aceternity/spotlight npx shadcn@latest add @aceternity/text-generate-effect npx shadcn@latest add @aceternity/background-beams npx shadcn@latest add @aceternity/tracing-beam # 21st.dev npx shadcn@latest add https://21st.dev/r/aceternity/text-hover-effect

4. Required utility (cn)

All components import cn() from @/lib/utils. Create if missing:

// lib/utils.ts import { type ClassValue, clsx } from "clsx" import { twMerge } from "tailwind-merge" export function cn(...inputs: ClassValue[]) { return twMerge(clsx(inputs)) } // npm install clsx tailwind-merge
