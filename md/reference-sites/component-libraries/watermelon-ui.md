# Watermelon UI — Component Reference

> Source: https://ui.watermelon.sh
> Style: Fresh, bold, colorful — standout interactive components
> Use for: Unique interactions, bold CTAs, eye-catching UI elements

---

## What It Is
Watermelon UI offers distinctive, high-creativity components with bold interactions. Less common than shadcn/aceternity — gives sites a unique feel. Good for brands wanting to stand out.

## How to Browse
1. Go to https://ui.watermelon.sh
2. Each component is previewed with live interaction
3. Copy the React + Tailwind source

## Brand Applications

### GardenSuites4You (yellow/navy)
- Bold CTA buttons with playful interactions
- Unique hover states for service cards
- Eye-catching counter animations for ROI stats

### DJ Custom Reno (amber/black)
- Bold before/after interaction components
- Standout quote request form styling
