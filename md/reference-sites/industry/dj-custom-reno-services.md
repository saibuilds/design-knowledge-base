### 2. Cabinet Spray Finishing — Core Differentiator

#### What DJ Custom Reno Does
- **New cabinet door spraying** — spray any door to any colour/sheen
- **Cabinet refinishing / repainting** — existing cabinets stripped, prepped, professionally resprayed
- **On-site kitchen cabinet repaints** — doors removed, sprayed off-site, reinstalled
- **B2B spraying for other cabinet shops** — shops send raw/primed doors, we spray and return

#### Finish Types Offered
| Finish | Description | Best For |
|--------|-------------|----------|
| **Lacquer (CV / Pre-cat)** | Conversion varnish — hardest, most durable; resists heat, moisture, yellowing | All kitchen doors, high-traffic |
| **Water-based paint** | Low VOC, fast dry, crisp colour match; Benjamin Moore / SW / RAL | White shaker, painted MDF |
| **Oil-based alkyd** | Self-leveling, very hard film; slight warm tone | Classic kitchens, clients wanting oil finish |
| **Solid wood stain + topcoat** | Wiping/gel stain → CV lacquer or waterborne poly topcoat | Oak, maple, walnut, cherry doors |
| **Two-tone** | Different colour uppers vs lowers, island accent | 2024–2026 trend, modern kitchens |

#### Spray Process
1. Strip / degrease existing finish
2. Repair chips, fill grain on MDF edges
3. Prime — high-build, sand 220 grit
4. 2× colour coats — HVLP spray booth
5. QC check → package → deliver or reinstall

#### B2B Shop Services
- Accept raw MDF, primed MDF, solid wood doors in any profile
- Spray to spec: colour code + sheen level
- Turnaround: 3–5 business days (rush available)
- Volume pricing for 50+ doors/week

### 5. Garden Suite / ADU Construction

#### What's Included
- Full turnkey: design, permit, build, utility connection
- Types: detached garden suite (backyard), laneway suite, garage conversion
- Suite sizes: 400–1,200 sq ft
- Structural: wood frame or modular panel system
- Finishes: same quality as primary home (not prefab box)

#### Permit Process (Toronto)
1. Zoning check (lot size, setbacks, lot coverage)
2. Site plan + architectural drawings
3. Building permit application → City review (8–16 weeks)
4. Inspections: framing, insulation, rough-in, final
5. Occupancy certificate

#### Garden Suite Business Case
- Average build cost: $180,000–$350,000 (size/finish dependent)
- Average Toronto rental income: $2,200–$3,200/month
- ROI: `(annual rent / build cost) × 100` → typically 8–15%
- Payback period: `build cost / annual rent` → typically 7–12 years
- Increases property value: $150K–$300K assessed value addition
