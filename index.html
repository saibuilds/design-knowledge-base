Ready-made prompts that reference this knowledge base. Use these as starting points when building components with Claude.

Hero Section
Build a dark hero section for a renovation company. Reference ~/design-knowledge-base/. Use DJ Custom Reno's color palette (#010101 bg, #2a53e8 accent). Add a GSAP word-reveal animation on load and a subtle Lenis scroll parallax on the background image. Use shadcn/ui Button component.
Scroll Animation
Build a services section that uses GSAP ScrollTrigger to stagger-reveal cards as they enter the viewport. Reference ~/design-knowledge-base/animations/gsap.html for the stagger pattern. Use Lenis smooth scroll — reference ~/design-knowledge-base/animations/lenis.html for the Next.js provider setup.
3D Hero with R3F
Build a 3D hero section using React Three Fiber. Reference ~/design-knowledge-base/3d/r3f.html. Add a floating torus knot that rotates based on mouse position. Use dynamic import with ssr:false for Next.js compatibility. Overlay centered headline text above the canvas.
Component Library
Build a stats/metrics section using shadcn/ui Card components. Reference ~/design-knowledge-base/components/shadcn.html for the dark theme CSS variables. Add Border Beam animation from Magic UI (~/design-knowledge-base/components/magicui.html) on each card. Animate numbers with GSAP counter pattern.
Page Transition
Add Framer Motion page transitions to a Next.js 14 App Router project. Reference ~/design-knowledge-base/animations/framer-motion.html for the AnimatePresence pattern. Use a full-screen wipe overlay (black, 0.5s ease). Wrap the root layout children with the transition component.
Full Site Bootstrap
Bootstrap a Next.js 15 renovation company site. Reference ~/design-knowledge-base/frameworks/nextjs.html for setup. Use Inter + Syne fonts with the Tailwind CSS variable pattern shown. Add shadcn/ui, Lenis, and GSAP. Set up metadata for SEO using the static metadata pattern. Dark theme with amber accent (#f59e0b).
Awwwards-Style Nav
Build a full-screen overlay navigation menu. Reference ~/design-knowledge-base/inspiration/awwwards-patterns.html for the GSAP clip-path reveal pattern. Links should stagger-animate in on open. Include a close button. Add smooth mouse-reactive position shift on the menu background (parallax on mouse move).
GLSL Shader Background
Build a full-viewport animated GLSL shader background using React Three Fiber. Reference ~/design-knowledge-base/3d/shaders.html for the shaderMaterial + Drei pattern and the noise distortion vertex shader. Use uTime uniform for animation. Render a plane geometry that fills the screen behind page content.
