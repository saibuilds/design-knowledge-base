Typography Scale
  • Font: Inter (their brand font, they literally use it as baseline)
  • Hero H1: clamp(40px, 6vw, 72px) — weight 600, letter-spacing -0.03em, line-height 1.05
  • Section H2: 36–48px, weight 600, -0.025em tracking
  • Feature headline: 20–24px, weight 500, -0.01em
  • Body text: 16–17px, weight 400, line-height 1.65, color rgba(255,255,255,.6)
  • Labels/captions: 12–13px, weight 500, letter-spacing 0.02em, all caps optional
  • Key insight: tight letter-spacing (-0.03em) on large headings, slightly looser (0) on body — creates hierarchy through tracking alone
Micro-Interactions

Button States

  • Primary btn: bg #5e6ad2, hover darkens slightly + translateY(-1px)
  • Ghost btn: border rgba(255,255,255,.1), hover fills with rgba(255,255,255,.05)
  • Active: scale(0.97) for 100ms on press
  • Focus: 2px offset outline in accent color
  • Transition: 150ms ease — extremely snappy

Navigation

  • Sticky nav with backdrop-blur + subtle border on scroll
  • Nav items: hover state fills with bg pill that slides in
  • Dropdown menus: scale(0.97) → scale(1), opacity 0→1, origin top
  • Active section indicator via scroll spy
  • Mobile: slide-in panel, no jarring transitions

Feature Cards

  • Card hover: border opacity increases from 0.06 to 0.15
  • Inner glow on hover: box-shadow with accent color at low opacity
  • Icon/graphic animates on card hover
  • No dramatic scale — 1:1 with glow instead

Scroll Animations

  • Sections fade + slide in from y:20 to y:0
  • IntersectionObserver-based — no GSAP dependency
  • Stagger: 0.1s between sibling elements
  • Product screenshots have parallax tilt effect
Layout Patterns
  • Max-width container: 1200px, centered, 40px horizontal padding
  • Hero layout: text left, product screenshot/graphic right (60/40 split)
  • Feature bento: asymmetric grid — 1 large card left, 2 stacked smaller right; or 3-col equal
  • Testimonial row: horizontal scrolling marquee of logos; below, 3-col testimonial cards
  • Pricing: 3-column with middle card elevated and highlighted (most popular)
  • Footer: minimal — 4-col links + logo, border-top only, no heavy decorative elements
  • Section rhythm: consistent 120px vertical spacing between sections
Motion Design Principles
  • Snappy, not dramatic: Linear's interactions are 100–200ms — not the 600ms theatrical animations of portfolio sites
  • Ease curves: ease-out for entrances (content arrives naturally), ease-in for exits, ease-in-out for state changes
  • Purpose-driven: every animation serves UX — confirms action, shows hierarchy, directs attention
  • No scroll jank: will-change: transform on animated elements, GPU-composited properties only (transform, opacity)
  • Gradient text animation: hero gradient text shifts angle/position with a subtle infinite animation (background-position loop)
  • Product demo animations: keyboard shortcuts appear, tasks complete, sprints update — feels alive without being noisy
What to Steal from Linear
  • Cool dark palette: The blue-tinted darks (#0f0f12, #1a1a24) feel more premium than neutral blacks — suggests precision
  • Gradient headline text: purple → violet → pink gradient on hero H1 — instantly eye-catching, now industry-standard for SaaS
  • Tight letter-spacing on headings: -0.03em on 60px+ text — looks professional, not cramped
  • Backdrop-blur nav: sticky nav with backdrop-filter: blur(12px) + semi-transparent bg — looks glass-like without heavy glassmorphism
  • 150ms transitions: Their snappy timing feels native and fast — avoid the slow 400ms+ transitions that feel sluggish in product UI
  • Active glow border: box-shadow: 0 0 0 1px rgba(94,106,210,0.4) on focused/active cards — subtle but clearly highlights selection

Animated Gradient Text Component

Create a React AnimatedGradientText component that displays text with a moving gradient background-clip. The gradient should be: linear-gradient(90deg, #5e6ad2, #a78bfa, #ec4899, #5e6ad2) at background-size: 200%. Animate background-position from 0% to -200% using a CSS keyframe animation on a 4s infinite loop. Text should be clipped to the gradient via -webkit-background-clip: text. Make it a simple wrapper: <GradientText className="text-6xl font-semibold">Your headline</GradientText>.

Code Snippet — Linear-Style Feature Card
// Linear-style feature card with glow hover effect
import { motion } from "framer-motion";

interface FeatureCardProps {
  icon: React.ReactNode;
  title: string;
  description: string;
  accentColor?: string;
}

export function FeatureCard({
  icon,
  title,
  description,
  accentColor = "#5e6ad2",
}: FeatureCardProps) {
  return (
    <motion.div
      className="group relative rounded-xl p-6 cursor-default"
      style={{
        background: "#1a1a24",
        border: "1px solid rgba(255,255,255,0.06)",
      }}
      whileHover={{
        borderColor: "rgba(255,255,255,0.12)",
        boxShadow: `0 0 0 1px ${accentColor}33, 0 8px 32px ${accentColor}15`,
      }}
      transition={{ duration: 0.15 }}
    >
      {/* Icon */}
      <div
        className="w-10 h-10 rounded-lg flex items-center justify-center mb-4 text-xl"
        style={{ background: `${accentColor}1a`, color: accentColor }}
      >
        {icon}
      </div>
      {/* Content */}
      <h3 className="text-white font-semibold text-[17px] tracking-[-0.01em] mb-2">
        {title}
      </h3>
      <p className="text-[rgba(255,255,255,0.5)] text-[14px] leading-[1.65]">
        {description}
      </p>
      {/* Subtle gradient shine on hover */}
      <div
        className="absolute inset-0 rounded-xl opacity-0 group-hover:opacity-100 transition-opacity duration-300 pointer-events-none"
        style={{
          background: `radial-gradient(400px at 50% 0%, ${accentColor}08, transparent)`,
        }}
      />
    </motion.div>
  );
}
