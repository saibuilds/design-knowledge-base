Lusion — Interactive Creative Studio

Creative Agency WebGL Three.js SOTD Multiple → lusion.co
Site Overview

Why Lusion Is the Benchmark for Creative Web

Lusion (London-based creative studio) is considered one of the most technically impressive agency sites on the internet. Multiple Awwwards SOTD and Site of the Year nominations. The site is a full-WebGL experience — nearly every surface, image, and transition is rendered through Three.js. It demonstrates that the gap between a "website" and an "interactive 3D experience" can be zero.

Scroll Behavior
  • Lenis smooth scroll: buttery inertia-based scrolling, dampening ~0.1, preventing jarring native scroll
  • Scroll-driven 3D scene: camera moves through a 3D world as the user scrolls — no separate "sections" per se, one continuous journey
  • Parallax depth layers: foreground, midground, background elements move at different rates creating real depth
  • ScrollTrigger pinning: certain scenes are pinned while the 3D content within them animates on scroll progress
  • Velocity-sensitive: faster scrolling increases motion blur / distortion effects in the WebGL pass
  • No scroll jank: everything runs on the GPU, no layout thrashing — maintains 60fps even on complex scenes
WebGL Use

Rendering Stack

  • Three.js as the WebGL layer
  • Custom GLSL vertex + fragment shaders
  • Post-processing: bloom, chromatic aberration, film grain
  • EffectComposer for multi-pass rendering
  • Environment maps (HDRI) for realistic reflections
  • Draco-compressed GLTF models for fast loading

Shader Techniques

  • Image distortion shader — mouse position warps textures
  • Displacement maps on image planes for depth illusion
  • Fresnel rim lighting on 3D objects
  • Custom noise functions for organic movement
  • Shader-based particle systems (100k+ particles)
  • Transition shaders: wave distortion between pages
Typography
  • Font: Custom variable font or PP Neue Montreal — geometric grotesque, clean, modern
  • Hero headline: 100–140px, weight 500–700, letter-spacing -0.02em, off-white (#f0ede8)
  • Body text: 16–18px, line-height 1.6, lighter weight (300–400)
  • Navigation: tiny — 12–13px, all caps, letter-spacing 0.12em, top-right alignment
  • Text as texture: typography placed over 3D scenes, often with blend modes (multiply, screen)
  • Animated type: text lines reveal with clip-path or transform on scroll, staggered per line

Animation / Interaction Breakdown

Mouse Tracking

  • Every object subtly follows mouse
  • Lag applied via lerp() each frame
  • Parallax on depth layers
  • Cursor is a custom dot

Hover States

  • Project images tilt in 3D (CSS perspective)
  • WebGL distortion on image hover
  • Text elements shift slightly
  • Color temperature shift

Page Load

  • Assets preloaded before display
  • Percentage counter (0–100%)
  • Reveal animation after load
  • Staggered content entrance
What to Steal from Lusion
  • Lerp-smoothed mouse tracking: Apply to any element — cursor, cards, hero images. Just 2 lines: currentX += (targetX - currentX) * 0.08 each rAF tick
  • Warm near-black palette: Use #0d0c0b instead of #000000, #f0ede8 instead of #ffffff — instantly feels more premium
  • Film grain post-process: Add SVG noise filter or canvas grain pass over the entire page — adds texture without complexity
  • Velocity-based distortion: On fast scroll, briefly increase a shader uniform or CSS filter blur, then ease back — suggests physical momentum
  • Loading preloader with counter: Show asset loading progress as a % counter, then do a reveal animation — sets expectation and feels intentional
  • Off-white text instead of white: #f0ede8 or #e8e4de on dark bg has warmth that #ffffff lacks

Lusion-Inspired R3F Hero with Mouse Parallax

Build a React Three Fiber hero section inspired by Lusion. The scene should: (1) render a floating 3D object (icosahedron or torus knot) with a warm metallic material (roughness 0.2, metalness 0.8) lit by a warm point light (#e8a87c) and a cool ambient, (2) apply mouse-tracking parallax — lerp the camera position toward mouse coordinates each frame (lerpFactor: 0.05), (3) add post-processing with @react-three/postprocessing: bloom (intensity 0.8, luminanceThreshold 0.6), chromatic aberration (offset 0.002), film grain (intensity 0.3), (4) overlay HTML text on top of the canvas using R3F Html component or absolute positioning — font off-white #f0ede8, 100px display size, letter-spacing -0.02em. Background color: #0d0c0b.

GLSL Image Distortion on Hover

Create a Three.js image plane with a custom GLSL fragment shader that distorts the texture on hover. The shader should: use a simplex noise function to offset UV coordinates, controlled by a 'distortion' uniform (0.0 at rest, 1.0 on hover). Animate the uniform with GSAP (0 → 1 on mouseenter, 1 → 0 on mouseleave, duration 0.8s ease). The distortion magnitude should be 0.03 at peak — subtle but noticeable. Apply the effect as a Three.js PlaneGeometry with ShaderMaterial. The plane should replace a standard img tag in the portfolio grid.

Code Snippet — Lerp Mouse Tracking
// Lusion-style lerp mouse tracking — smooth parallax on any element
import { useEffect, useRef } from "react";

export function useLerpMouse(factor = 0.08) {
  const current = useRef({ x: 0, y: 0 });
  const target = useRef({ x: 0, y: 0 });
  const raf = useRef<number>(0);

  useEffect(() => {
    const onMove = (e: MouseEvent) => {
      // Normalize to -1 to 1
      target.current.x = (e.clientX / window.innerWidth) * 2 - 1;
      target.current.y = (e.clientY / window.innerHeight) * 2 - 1;
    };
    window.addEventListener("mousemove", onMove);

    const tick = () => {
      current.current.x += (target.current.x - current.current.x) * factor;
      current.current.y += (target.current.y - current.current.y) * factor;
      raf.current = requestAnimationFrame(tick);
    };
    raf.current = requestAnimationFrame(tick);

    return () => {
      window.removeEventListener("mousemove", onMove);
      cancelAnimationFrame(raf.current);
    };
  }, [factor]);

  return current; // current.x, current.y — use in useFrame or rAF
}

// CSS film grain overlay — Lusion signature texture
// Add this component once at the root, wrapping the entire page

export function FilmGrain() {
  return (
    <>
      {/* SVG noise filter defined once */}
      <svg className="hidden">
        <defs>
          <filter id="grain">
            <feTurbulence
              type="fractalNoise"
              baseFrequency="0.65"
              numOctaves="3"
              stitchTiles="stitch"
            />
            <feColorMatrix type="saturate" values="0" />
          </filter>
        </defs>
      </svg>
      {/* Animated grain overlay */}
      <div
        className="pointer-events-none fixed inset-0 z-[9999]"
        style={{
          opacity: 0.035,
          filter: "url(#grain)",
          background: "#fff",
          animation: "grain 0.5s steps(1) infinite",
        }}
      />
      <style>{`
        @keyframes grain {
          0%, 100% { transform: translate(0, 0); }
          10% { transform: translate(-2%, -3%); }
          20% { transform: translate(1%, 2%); }
          30% { transform: translate(-3%, 1%); }
          40% { transform: translate(2%, -2%); }
          50% { transform: translate(-1%, 3%); }
          60% { transform: translate(3%, -1%); }
          70% { transform: translate(-2%, 2%); }
          80% { transform: translate(1%, -3%); }

      `}</style>
    </>
  );
}
