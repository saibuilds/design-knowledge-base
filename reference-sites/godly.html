What Godly Is

Godly is a curated gallery of the most beautiful, innovative websites on the internet. Unlike Awwwards, Godly focuses purely on aesthetic excellence — no scoring system, just hand-picked sites that represent the cutting edge of web design. The site itself is a masterclass in minimal dark UI with impeccable grid layout and hover animations. It's categorized by: Agency, Portfolio, Startup, E-commerce, Editorial, and Interactive. Updated daily with 3–5 new submissions.

Godly's Own Design System

Layout

  • Pure CSS grid — typically 3 columns on desktop, 2 on tablet, 1 on mobile
  • Consistent 8px gap between cards
  • Cards maintain 16:10 aspect ratio
  • Minimal chrome — no distracting UI
  • Top filter bar with category pills
  • Infinite scroll pagination (no pagination buttons)

Hover Interactions

  • Card scale: 1 → 1.02 on hover
  • Live video preview plays on hover (no sound)
  • Overlay with site name fades in from bottom
  • Smooth cubic-bezier transitions: cubic-bezier(.16,1,.3,1)
  • Screenshot → video crossfade within 0.3s
  • Category tag visible on hover
Color Palette

Typography
  • Font: custom or system-ui — extremely minimal, body 13–14px
  • Site name on cards: 13px, weight 500, white
  • Category tags: 11px, weight 400, muted gray
  • Header logo: minimal wordmark, no decorative elements
  • Filter pills: 12px, padding 4px 12px, border-radius 20px
  • Zero decorative typography — all utility
Featured Design Styles Seen on Godly
  • Brutalist minimalism: No gradients, raw black/white, heavy type, intentional misalignment
  • Immersive WebGL: Sites where the browser window becomes a canvas — particles, fluid sims, 3D worlds
  • Scroll storytelling: Narrative unfolds as you scroll — text reveals, scene changes, character movement
  • Kinetic type-first: Zero images, pure typography as the visual element — letters scale, rotate, flow
  • Gradient meshes: Soft, multi-stop CSS/SVG gradient backgrounds with organic shape movement
  • Photography-forward: Full-bleed editorial photography with minimal UI overlay
  • Retro-futurism: CRT effects, scan lines, green-on-black terminal aesthetics, pixel fonts

Animation Patterns
  • Staggered card entrance: Cards fade + translate-y on page load, stagger 0.04s between items
  • Video hover preview: muted autoplay video crossfades over static screenshot on mouseenter
  • Filter pill transition: Active pill has bg fill that slides in with layout animation
  • Infinite scroll: New cards append with fade-in as IntersectionObserver triggers near bottom
  • Smooth hover scale: transform: scale(1.02) with will-change: transform for GPU compositing
What to Steal
  • Gallery grid pattern: 3-col CSS grid with consistent aspect-ratio cards — works for any showcase page
  • Video-on-hover preview: Replaces static thumbnails with video — dramatically increases engagement
  • Near-black multi-tone bg: Using #0c, #16, #22 for layered depth without using true gradients
  • Borderless card separation: Cards defined only by background color difference, no border lines
  • Category filter with scroll-persist: Filter state preserved in URL hash so shareable links work

Godly-Style Design Gallery

Build a Godly-inspired design gallery in Next.js. Features: (1) 3-column responsive CSS grid with 16:10 aspect-ratio cards, 8px gap, bg #0c0c0c background, (2) on hover, a muted autoplay video fades in over the thumbnail image with scale(1.02) transition using cubic-bezier(0.16,1,0.3,1) 0.3s, (3) overlay with site name (13px white 500) and category tag (11px #999) fades up from bottom on hover, (4) category filter pills at top — clicking one filters the grid with Framer Motion layout animations so cards smoothly reflow, (5) infinite scroll using IntersectionObserver, new cards appear with staggered fade+translateY entrance.

Video-Preview Card Component

Create a React card component that shows a static screenshot by default and plays a muted looping video on hover. Use CSS transition for the crossfade (opacity on the video element: 0 → 1 over 0.3s). The card maintains a fixed aspect-ratio via aspect-ratio: 16/10. Include a bottom overlay div that slides up from y:8 to y:0 on hover showing the site title and category. Make it accessible: add title attribute and role="button" with keyboard support.

Code Snippet — Godly-Style Gallery Grid
// Godly-style gallery with video hover preview
"use client";
import { useState, useRef } from "react";
import { motion } from "framer-motion";
import Image from "next/image";

interface GalleryItem {
  id: string;
  title: string;
  category: string;
  thumbnail: string;
  video?: string;
}

function GalleryCard({ item }: { item: GalleryItem }) {
  const [hovered, setHovered] = useState(false);
  const videoRef = useRef<HTMLVideoElement>(null);

  const handleEnter = () => {
    setHovered(true);
    videoRef.current?.play();
  };
  const handleLeave = () => {
    setHovered(false);
    if (videoRef.current) { videoRef.current.pause(); videoRef.current.currentTime = 0; }
  };

  return (
    <motion.div
      className="relative overflow-hidden rounded-lg bg-[#161616] cursor-pointer"
      style={{ aspectRatio: "16/10" }}
      onMouseEnter={handleEnter}
      onMouseLeave={handleLeave}
      whileHover={{ scale: 1.02 }}
      transition={{ duration: 0.3, ease: [0.16, 1, 0.3, 1] }}
    >
      <Image src={item.thumbnail} alt={item.title} fill className="object-cover" />
      {item.video && (
        <video
          ref={videoRef}
          src={item.video}
          muted
          loop
          playsInline
          className="absolute inset-0 w-full h-full object-cover transition-opacity duration-300"
          style={{ opacity: hovered ? 1 : 0 }}
        />
      )}
      {/* Bottom overlay */}
      <motion.div
        className="absolute bottom-0 left-0 right-0 p-3 bg-gradient-to-t from-black/80 to-transparent"
        initial={{ opacity: 0, y: 8 }}
        animate={{ opacity: hovered ? 1 : 0, y: hovered ? 0 : 8 }}
        transition={{ duration: 0.2 }}
      >
        <p className="text-white text-[13px] font-medium">{item.title}</p>
        <p className="text-[#999] text-[11px]">{item.category}</p>
      </motion.div>
    </motion.div>
  );
}

export function GodlyGallery({ items }: { items: GalleryItem[] }) {
  return (
    <div className="min-h-screen bg-[#0c0c0c] p-4">
      <motion.div
        className="grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 gap-2"
      >
        {items.map((item, i) => (
          <motion.div
            key={item.id}
            initial={{ opacity: 0, y: 16 }}
            animate={{ opacity: 1, y: 0 }}
            transition={{ delay: i * 0.04, duration: 0.4 }}
          >
            <GalleryCard item={item} />
          </motion.div>
        ))}
      </motion.div>
    </div>
  );
}
