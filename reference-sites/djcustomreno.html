Improvement Notes

Performance — Rebuild in Next.js
Current WordPress/Elementor stack loads 4-6MB of JS. A Next.js rebuild with static generation would achieve sub-1s LCP. Use ISR for dynamic content (gallery, testimonials).
Hero — Add scroll-triggered text reveal
Current hero is static image + text. Adding a GSAP timeline on load (nav fade in → headline reveal → subtext → CTA button) would dramatically increase perceived quality. Use lenis for smooth scroll throughout.
Services — Replace static cards with animated grid
Use a bento grid layout (Magic UI) with stagger-in scroll animation. Each service card gets an icon, micro-animation on hover (border beam), and links to the service detail page.
Gallery — Before/After slider
Aceternity's Compare Slider component is perfect here. Before/after drag comparison with smooth CSS transition. Add a filterable masonry grid for the full portfolio.
Trust signals — Animated stats section
Add a stats bar with animated counters (GSAP or Magic UI NumberTicker): "200+ Projects Completed", "15+ Years Experience", "98% Client Satisfaction". Trigger on scroll entry.
Typography — Upgrade to Syne + Inter pairing
Replace the current generic font with Syne 800 for headings (bold, distinctive) and Inter 400 for body. This pairing signals premium without being cold.
Mobile — Fix the nav menu
The current mobile menu is a basic WordPress dropdown. Replace with an Aceternity FloatingNav or a full-screen overlay menu with staggered link animation. Use Sheet from shadcn for the overlay.
