Stripe — Gradients, Transitions & Typography

Payments Platform Gradient Mastery Section Transitions Typography Excellence → stripe.com

Financial infrastructure for the internet.

Millions of companies of all sizes use Stripe to accept payments, send payouts, and manage their businesses online.

Color Palette

Stripe's palette is deep indigo/violet — rich, professional, not generic "tech blue". Their gradient style is diagonal with multiple warm-to-cool stops:

Gradient Techniques

Hero Background Gradient

  • Multi-stop diagonal linear-gradient (135deg) for base
  • Layered radial-gradient blobs for glow spots
  • Colors flow: deep navy → indigo → violet → magenta
  • CSS transition between gradient stops on scroll (JS-driven)
  • Never pure flat color — always slight gradient even in "plain" sections
  • Animated: gradient position shifts slowly via keyframe loop

Section Gradient Transitions

  • Adjacent sections "blend" via overlapping gradients on their borders
  • Section bottom has a gradient fade to the next section's bg color
  • SVG wave dividers with gradient fill instead of sharp horizontal lines
  • Background color gradients from #0d0d2b (hero) to #fff (middle) to #f6f9fc (features)
  • The scroll experience feels like moving through one continuous gradient world
Typography
  • Font: Sohne (custom licensed) — geometric, strong, modern sans-serif. Fallback: system-ui or "Helvetica Neue"
  • Hero headline: 60–80px, weight 700, letter-spacing -0.025em, line-height 1.05 — tight and impactful
  • Subheadline: 20–24px, weight 400–500, color rgba(255,255,255,.65), line-height 1.6
  • Feature section headings: 34–40px, weight 600, dark color #0a2540 (on white sections)
  • Body: 17–18px, line-height 1.7, color #425466 (on white) or rgba(255,255,255,.7) (on dark)
  • All-caps labels: 13px, weight 600, letter-spacing 0.1em — used for category/section labels
  • Stripe's type hallmark: extremely generous line-height + comfortable measure (max 65ch) for readability
Section Transitions
  • Wave SVG dividers: Custom wave shapes between sections using SVG path with gradient fill — never hard horizontal cuts
  • Gradient dissolve: Section backgrounds fade into each other — implemented via overlapping ::before/::after with gradient-to-transparent
  • Color journey: Dark purple hero → light blue/white features → pure white middle sections → light gray footer — systematic progression
  • Sticky scroll animations: Product UI screenshots "build up" on scroll — rows of a dashboard appear one by one with ScrollTrigger
  • Parallax at section boundaries: Background graphic elements continue past their section boundary, creating depth overlap
Layout Patterns
  • Container: max-width 1080px, centered, 32px horizontal padding
  • Hero: centered text, large headline, subtitle, two CTAs, followed by a large product screenshot with gradient border/glow
  • Features alternating: Text left / UI screenshot right, then Text right / UI screenshot left — creates reading rhythm
  • Stats band: Full-width section with 3–4 large stats (e.g. "$1T+ processed") — simple grid, massive numbers
  • Testimonials: Card grid with company logo, quote, name/title — white cards on light gray bg
  • CTA section: Full-width dark gradient section with large headline + two buttons, used before footer
What to Steal from Stripe
  • Diagonal multi-stop gradients: bg: linear-gradient(135deg, #0d0d2b 0%, #1a1a4e 40%, #2d1b5e 70%, #1a0d3d 100%) — richer than 2-stop gradients
  • Wave section dividers: Replace horizontal lines with SVG waves — instantly elevates page polish
  • Radial glow blobs: Position large radial-gradient pseudo-elements off-center — creates the signature "Stripe glow"
  • Product screenshot as hero graphic: Your app UI in a browser frame with gradient glow — makes it feel real and aspirational
  • Color progression across page: Plan your sections to transition dark→light→dark — creates journey, not just sections
  • Generous type settings: line-height 1.7 for body text, max-width 65ch — dramatically improves readability

SVG Wave Section Divider

Create a React WaveDivider component that creates a smooth SVG wave transition between two colored sections. The component takes topColor and bottomColor props. The SVG path should create a gentle wave shape (1440x80 viewBox) with a gradient fill from topColor to bottomColor. Place it between sections using position: relative, margin-bottom: -1px to prevent seams. Offer a 'flip' prop that inverts the wave direction. This replaces all sharp section borders with organic transitions.

Code Snippet — Animated Gradient Background
// Stripe-style animated gradient background
// The gradient slowly shifts position using a CSS keyframe animation

export function AnimatedGradientBg({ children }: { children: React.ReactNode }) {
  return (
    <div className="relative overflow-hidden" style={{ background: "#0d0d2b" }}>
      {/* Animated gradient layer */}
      <div
        className="absolute inset-0 pointer-events-none"
        style={{
          background: `
            linear-gradient(135deg, #0d0d2b 0%, #1a1a4e 40%, #2d1b5e 70%, #1a0d3d 100%)
          `,
          animation: "gradientShift 8s ease infinite",
          backgroundSize: "200% 200%",
        }}
      />
      {/* Glow blob 1 — indigo */}
      <div
        className="absolute pointer-events-none"
        style={{
          top: "-20%", left: "-10%",
          width: "60%", height: "80%",
          background: "radial-gradient(ellipse, rgba(99,102,241,0.3) 0%, transparent 60%)",
        }}
      />
      {/* Glow blob 2 — pink */}
      <div
        className="absolute pointer-events-none"
        style={{
          bottom: "-20%", right: "-10%",
          width: "50%", height: "70%",
          background: "radial-gradient(ellipse, rgba(236,72,153,0.2) 0%, transparent 60%)",
        }}
      />
      {/* Content */}
      <div className="relative z-10">{children}</div>
      <style>{`
        @keyframes gradientShift {
          0% { background-position: 0% 50%; }
          50% { background-position: 100% 50%; }
          100% { background-position: 0% 50%; }
        }
      `}</style>
    </div>
  );
}

// Wave Divider Component
export function WaveDivider({ topColor = "#0d0d2b", bottomColor = "#fff", flip = false }) {
  return (
    <div style={{ marginBottom: -1, lineHeight: 0 }}>
      <svg
        viewBox="0 0 1440 80"
        xmlns="http://www.w3.org/2000/svg"
        preserveAspectRatio="none"
        style={{ display: "block", width: "100%", transform: flip ? "scaleY(-1)" : "none" }}
      >
        <defs>
          <linearGradient id="waveGrad" x1="0" x2="0" y1="0" y2="1">
            <stop offset="0%" stopColor={topColor} />
            <stop offset="100%" stopColor={bottomColor} />
          </linearGradient>
        </defs>
        <path
          d="M0,40 C360,80 1080,0 1440,40 L1440,80 L0,80 Z"
          fill="url(#waveGrad)"
        />
      </svg>
    </div>
  );
}
