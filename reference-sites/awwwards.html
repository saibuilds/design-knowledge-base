What Awwwards Is

Awwwards is the premier web design award platform judging sites on Design (40%), Usability (20%), Creativity (20%), and Content (20%). Sites receive nominations from the community and are judged by a panel. Top honors: Site of the Day (SOTD), Developer Award, Honorable Mention. The platform features over 30,000 awarded sites spanning agency portfolios, product sites, editorial, interactive experiments, and immersive 3D experiences.

Common Design Patterns in Award Winners

Full-Screen Hero + WebGL

Immersive entry with 3D canvas, particle systems, or distortion shaders covering the entire viewport.

SOTD PATTERN

Horizontal Scroll Sections

GSAP-powered horizontal scroll panels for case studies, timelines, or portfolio pieces.

DEVELOPER AWARD PATTERN

SVG Path Morphing

Animated SVG shapes that morph on scroll or mouse move using GSAP MorphSVG or CSS animations.

HONORABLE MENTION PATTERN

Kinetic Typography

Large type that animates letter-by-letter, splits into lines, or tracks cursor movement.

SOTD PATTERN

Image Grid Hover Distortion

CSS or shader-based image distortion on hover, often with cursor-following displacement maps.

COMMON PATTERN

INTERACTION PATTERN

Typography Patterns

Editorial / Agency Style

  • Variable fonts with weight/width animation on scroll
  • Huge display sizes: 120–200px headers
  • Mixed serif + grotesque pairings
  • Popular: Neue Haas Grotesk, Editorial New, PP Neue Montreal
  • Tight leading: line-height 0.85–0.95 for headlines

Product / SaaS Style

  • Clean sans-serifs: Inter, Geist, Sohne
  • Gradient text on hero headlines
  • Mono accents for code/technical labels
  • Precise type scales: 12/14/16/20/32/48/64px
  • Letter-spacing: -0.02em to -0.04em on large headings
Animation / Interaction Notes
  • Page transitions: Full-page overlay wipes (clip-path or GSAP cover panels) are near-universal in top-scoring sites
  • Scroll-linked: Progress bar, parallax depth (foreground/background layers), pinned scroll scenes
  • Cursor: Custom cursor is expected — circle follower, magnetic snap, blend-mode inversion
  • Load animation: Counter loading screen, logo reveal, staggered text entrance
  • Image hover: 3D tilt (CSS perspective + JS mouse tracking), displacement maps via WebGL
  • Sound: Optional ambient audio toggle (click-on-demand, not autoplay)
What Makes Sites Win
  • Design (40%): Visual hierarchy, whitespace, color harmony, consistent grid, original aesthetic
  • Usability (20%): Navigation clarity, loading speed, accessible interactions, mobile adaptation
  • Creativity (20%): Unexpected layouts, novel interactions, conceptual coherence, uniqueness
  • Content (20%): Copy quality, imagery resolution, video direction, storytelling arc
  • Key differentiators: Smooth 60fps animations, no janky scroll, purposeful micro-interactions, cohesive brand language throughout
Techniques Worth Stealing
  • Split text entrance: Split headline into chars/words, stagger animate each with GSAP or Framer Motion (y: 100 → 0, opacity 0→1)
  • Scroll-scrubbed background color: Transition bg color between sections using ScrollTrigger scrub
  • Marquee ticker: Infinite scrolling text band used as section dividers — adds motion without complexity
  • Noise grain overlay: CSS pseudo-element with SVG feTurbulence filter or a tileable PNG at low opacity (~0.04)
  • Magnetic buttons: Track mouse distance, translate button toward cursor within 80px radius using gsap.to()
  • Variable font weight on scroll: Animate font-variation-settings weight from 100 to 900 as user scrolls
Claude Prompts to Recreate Awwwards-Style Patterns

Awwwards-Style Hero with Split Text + Cursor

Build an Awwwards-quality hero section in React with: (1) split text entrance where each word animates from y:60 to y:0 with opacity 0→1, staggered 0.08s using Framer Motion, (2) a custom magnetic cursor — a 24px circle that follows mouse with spring physics and snaps to buttons within 80px, changing blend mode to 'difference' on hover, (3) full-viewport dark background #0a0a0a, (4) noise grain overlay using a CSS filter SVG feTurbulence pseudo-element at 0.04 opacity. Font: Neue Haas Grotesk or fallback system-ui, 120px weight 700, letter-spacing -0.03em.

Horizontal Scroll Portfolio Section

Create a GSAP horizontal scroll section for a portfolio. Pin the section for 400vh of vertical scroll. Translate the inner track from x:0 to x:-80% based on scroll progress using ScrollTrigger scrub:1. Each project card is 500px wide with an image, project name in large type, and category tag. On hover, the image scales to 1.05 and a color overlay fades in. Add a progress bar at the bottom tracking scroll position within this section.

Page Transition with Cover Panel

Build a page transition in Next.js App Router using a full-viewport cover panel. On link click: a dark panel (#111) animates from scaleY:0 (bottom) to scaleY:1 covering the screen (0.4s ease), then the new page content fades in as the panel animates out to scaleY:0 from the top. Use Framer Motion AnimatePresence with layout-aware exit animations. The panel should use transform-origin: bottom on enter and top on exit.

Code Snippet — Awwwards Split Text Entrance
// Tailwind + Framer Motion — Split text word-by-word entrance
import { motion } from "framer-motion";

const container = {
  hidden: {},
  visible: { transition: { staggerChildren: 0.08 } }
};

const word = {
  hidden: { y: 60, opacity: 0 },
  visible: { y: 0, opacity: 1, transition: { duration: 0.65, ease: [0.16, 1, 0.3, 1] } }
};

export function SplitHero({ text }: { text: string }) {
  const words = text.split(" ");
  return (
    <section className="relative min-h-screen bg-[#0a0a0a] flex items-center overflow-hidden">
      {/* Noise grain overlay */}
      <div className="pointer-events-none absolute inset-0 opacity-[0.04]"
        style={{ backgroundImage: "url(\"data:image/svg+xml,%3Csvg viewBox='0 0 200 200' xmlns='http://www.w3.org/2000/svg'%3E%3Cfilter id='n'%3E%3CfeTurbulence type='fractalNoise' baseFrequency='0.9' numOctaves='4'/%3E%3C/filter%3E%3Crect width='100%25' height='100%25' filter='url(%23n)'/%3E%3C/svg%3E\")", backgroundSize: "200px" }}
      />
      <div className="px-8 md:px-16">
        <motion.h1
          className="text-[clamp(48px,10vw,140px)] font-bold tracking-[-0.03em] text-white leading-[0.9] flex flex-wrap gap-x-[0.25em]"
          variants={container}
          initial="hidden"
          animate="visible"
        >
          {words.map((w, i) => (
            <div key={i} className="overflow-hidden">
              <motion.span className="block" variants={word}>{w}</motion.span>
            </div>
          ))}
        </motion.h1>
        <motion.p
          className="mt-8 text-[#737373] text-lg max-w-xl"
          initial={{ opacity: 0, y: 20 }}
          animate={{ opacity: 1, y: 0 }}
          transition={{ delay: 0.6, duration: 0.5 }}
        >
          Award-winning design starts with purposeful motion.
        </motion.p>
      </div>
    </section>
  );
}
Code Snippet — Magnetic Button
// Magnetic button — attracts toward cursor within proximity radius
import { useRef } from "react";
import { motion, useMotionValue, useSpring } from "framer-motion";

export function MagneticButton({ children }: { children: React.ReactNode }) {
  const ref = useRef<HTMLButtonElement>(null);
  const x = useMotionValue(0);
  const y = useMotionValue(0);
  const sx = useSpring(x, { stiffness: 200, damping: 20 });
  const sy = useSpring(y, { stiffness: 200, damping: 20 });

  const handleMouseMove = (e: React.MouseEvent) => {
    const btn = ref.current!;
    const rect = btn.getBoundingClientRect();
    const cx = rect.left + rect.width / 2;
    const cy = rect.top + rect.height / 2;
    const dist = Math.hypot(e.clientX - cx, e.clientY - cy);
    if (dist < 80) {
      x.set((e.clientX - cx) * 0.35);
      y.set((e.clientY - cy) * 0.35);
    }
  };

  const handleMouseLeave = () => { x.set(0); y.set(0); };

  return (
    <motion.button
      ref={ref}
      style={{ x: sx, y: sy }}
      onMouseMove={handleMouseMove}
      onMouseLeave={handleMouseLeave}
      className="px-8 py-4 border border-white/20 rounded-full text-white text-sm font-medium hover:bg-white hover:text-black transition-colors duration-300"
    >
      {children}
    </motion.button>
  );
}
