Cosmos.so — Gallery Layout & Card Design

Inspiration Gallery Masonry Layout Card Design Hover Effects → cosmos.so
What Cosmos Is

Cosmos (formerly Savee.it) is a visual inspiration board and discovery platform for designers and creatives. It's Pinterest for the design community — but with far better curation and a more minimal, tasteful UI. Users save images, screenshots, videos, and links into "boards." The discover feed is a masonry-style image gallery. The design of the platform itself is exemplary: dark, minimal, fast, and the images always take center stage. Key interaction pattern: everything is about the image — UI chrome is nearly invisible until needed.

Color Palette

Zero colored accents in the chrome. User-saved imagery and color-extracted dominant tones are the only color. This is intentional — the platform serves the content, not itself.

Gallery Layout Patterns

Masonry Grid (Main Feed)

  • CSS columns: 3–5 depending on viewport width
  • gap: 6–8px — tight packing
  • Images maintain natural aspect ratio (no forced crop)
  • Infinite scroll with virtualization for performance
  • On hover: black overlay fades in (opacity 0 → 0.5)
  • No border-radius on images — maximizes image area

Board Grid

  • Fixed-ratio grid for user boards (3-col, 1:1 forced)
  • First 3 images of a board shown as collage preview
  • Board title overlays bottom of the preview grid
  • Hover: slight scale(1.01) and border brightens
  • Count badge top-right on board cards
Hover Effects
  • Image overlay: Semi-transparent black (rgba(0,0,0,.5)) fades in on hover, revealing action buttons (save, share, open)
  • Action buttons: Circle icon buttons appear at bottom of overlay — white bg, black icon, scale(0.8)→scale(1) on overlay appear
  • Title reveal: Image title/source fades in at bottom of overlay if available
  • Cursor change: crosshair or grab cursor on images
  • No scale on images: Unlike many galleries, Cosmos does NOT scale images on hover — keeps layout stable, no gap-jumping
  • Transition speed: 200ms ease — fast, not theatrical
Card Design System
  • Card as image: The card IS the image — no white padding, no text below by default
  • Metadata on hover only: Source URL, tags, save count — all hidden until hover
  • Progressive loading: Blurred thumbnail placeholder (blurhash) before full image loads
  • Image format support: jpg, png, gif (plays on hover), mp4 (loops on hover), webp
  • Right-click menu: Custom context menu with save/board actions instead of native browser menu
  • Keyboard navigation: Arrow keys to navigate grid, Enter to open, Escape to close
What to Steal from Cosmos
  • CSS columns masonry: The simplest masonry — no JS library needed. columns: 3, column-gap: 8px, break-inside: avoid on items
  • Image-first card design: Let images fill 100% of the card — metadata in overlay, not below
  • Blurhash placeholders: Generate 20-char blurhash strings server-side, render as colored div before img loads — eliminates layout shift
  • No accent color in chrome: If your content is colorful, make the UI monochrome so it doesn't compete
  • Scale-free hover: Not every image needs to scale on hover — overlay-only hover is often cleaner
  • Tight gap: 6–8px between gallery items (not 16–24px) creates a denser, more immersive gallery feel
Claude Prompts

Cosmos-Style Masonry Gallery

Build a Cosmos-inspired masonry image gallery in React. Use CSS columns (not JS masonry) — 5 columns on xl, 4 on lg, 3 on md, 2 on sm, 1 on mobile. Column-gap and margin-bottom: 6px. Each card is an image with: (1) overflow-hidden, (2) break-inside: avoid, (3) position relative, (4) on hover: black overlay div (absolute inset-0 bg-black/50) fades in with opacity transition 200ms ease, (5) action buttons (save icon, open icon) appear in overlay bottom-right, (6) no scale animation. Images maintain natural aspect-ratio. Implement infinite scroll with an IntersectionObserver sentinel at the bottom. Background: #111111, no border-radius on images.

Board Preview Card Component

Create a BoardCard component inspired by Cosmos.so boards. The card shows a 3-image collage preview: a 2/3 width image on the left, two 1/3 width images stacked on the right, all fitting in a fixed-height container (200px). Below the collage: board title (14px weight-500 text-white), item count (12px text-gray-500). On hover: border brightens from rgba(255,255,255,0.06) to rgba(255,255,255,0.15), transition 200ms. The board card is a dark card (#1a1a1a) with rounded-xl. Clicking navigates to the board page.

Code Snippet — CSS Masonry Gallery
// Cosmos-style CSS masonry gallery with hover overlay
"use client";
import { useState } from "react";
import Image from "next/image";

interface GalleryImage {
  id: string;
  src: string;
  alt: string;
  width: number;
  height: number;
  title?: string;
}

function MasonryItem({ image }: { image: GalleryImage }) {
  const [hovered, setHovered] = useState(false);

  return (
    <div
      className="relative overflow-hidden cursor-pointer mb-[6px] break-inside-avoid"
      onMouseEnter={() => setHovered(true)}
      onMouseLeave={() => setHovered(false)}
    >
      <Image
        src={image.src}
        alt={image.alt}
        width={image.width}
        height={image.height}
        className="w-full h-auto block"
        style={{ display: "block" }}
      />
      {/* Hover overlay */}
      <div
        className="absolute inset-0 transition-opacity duration-200 flex items-end justify-between p-3"
        style={{
          background: "rgba(0,0,0,0.5)",
          opacity: hovered ? 1 : 0,
        }}
      >
        {image.title && (
          <span className="text-white text-xs font-medium">{image.title}</span>
        )}
        {/* Action buttons */}
        <div className="flex gap-2 ml-auto">
          <button
            className="w-8 h-8 rounded-full bg-white flex items-center justify-center text-black hover:scale-110 transition-transform"
            onClick={(e) => { e.stopPropagation(); /* save action */ }}
          >
            +
          </button>
          <button
            className="w-8 h-8 rounded-full bg-white flex items-center justify-center text-black hover:scale-110 transition-transform"
            onClick={(e) => { e.stopPropagation(); /* open action */ }}
          >
            ↗
          </button>
        </div>
      </div>
    </div>
  );
}

export function MasonryGallery({ images }: { images: GalleryImage[] }) {
  return (
    <div
      className="bg-[#111]"
      style={{
        columns: "5 220px",
        columnGap: "6px",
        padding: "6px",
      }}
    >
      {images.map((img) => (
        <MasonryItem key={img.id} image={img} />
      ))}
    </div>
  );
}
